# LEM

#WIP!!!!!



Linux Executable Manager, also know as LEM, is a powerful diverse package manager.
LEM is primarily made for Linux, but is supported by MacOS and other systems.

| System      | Supported |
| ----------- | :-------: |
| Linux       | ✅        |
| MacOS       | ✅        |
| FreeBSD     | ✅        |
| ChromeOS    | ✅        |
| Android     | ✅        |
| Windows     | ❌        |
